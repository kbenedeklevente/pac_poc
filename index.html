What to Look For

  • Vanilla: Open DevTools and watch the getImageData/putImageData dissolve — pixel-by-pixel corrosion that's impossible in a scene graph engine
  • PixiJS: Notice the DisplacementFilter "melting" effect on the panel — the GPU does the heavy lifting via shader displacement
  • Phaser: Navigate between rooms — the camera pans smoothly instead of cutting, and the dissolve uses a timeline with orchestrated tweens
  • All three: The puzzle chain is identical (beaker → pipe → panel → key → door) because shared/puzzleEngine.js drives all game logic
  • Architecture: Each main.js wires gameEvents to its renderer — the shared module never imports any engine code

How to Play

  • Click left/right screen edges or use arrow keys to navigate between rooms
  • Click objects to interact — pick up items, use them on targets
  • Click inventory slots to select an item, then click a target to use it
  • Puzzle: Find beaker (Storage) → fill at pipe (Lab) → pour acid on panel (Lab) → grab key → use on door (Corridor)
